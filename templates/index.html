Clothes Restapi

A small restapi of clothes.
Return json objects of a database made with sqlite3.

Routes

GET

  • Return list of clothes objects /clothes
  • Return a clothe using your id /clothe/1

POST

  • Save a new clothe in the database /add-clothe

PUT

  • Update a clothe with new values /update-clothe/1

DELETE

  • Delete a clothe of the database using your id /delete-clothe/1

Json to send

Use a json object for send data, necesary in case of POST and PUT request with the following keys.

Name

The name of the clothe

Price

The price for the clothe, must be a integer or decimal number

Size

The size for the clothe "XS", "S", "M", "L", "XL"
